# File-Naming-Conventions

File naming conventions are the unsung heroes of the video production industry.

# General Naming Convention

YYYY-MM-DD_TITLE

# Video Folder Structure

YYYY-MM_TITLE

- Root Folder
  - 01_Footage
  - 02_Audio
  - 03_Music
  - 04_Assets
  - 05_Exports

# Video File Naming Convention

PROJECT_SHOOT_###
